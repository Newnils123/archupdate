# archupdates
Full system update + cleanup (pacman, AUR, flatpak) for fish


# Installation 


save this file in :
`~/.config/fish/functions/upd.fish`

note : the file name is the command you need to execute in the terminal to call the function
ex : 
```fish
#!/usr/bin/env fish

~ $ upd
update official packages
[sudo] password for [your_user]: 
```

close and reopen the terminal
